## Raw Events Table

Individual user actions (Logins, Clicks, Purchases).

```sql events
select * from synthetic.raw_events
order by event_time desc
limit 500
```

<DataTable data={events} search=true rows=10>
    <Column id=event_id title="ID" />
    <Column id=event_time title="Timestamp" />
    <Column id=user_id title="User ID" />
    <Column id=event_type title="Action" />
    <Column id=revenue title="Revenue" fmt=currency />
</DataTable>
