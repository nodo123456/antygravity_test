# Data Explorer

Explore the raw datasets generated by the pipeline.

## 1. Customers Table

All users registered on the platform.

```sql customers
select * from synthetic.customers
limit 500
```

<DataTable data={customers} search=true rows=10>
  <Column id=user_id title="ID" />
  <Column id=name title="Name" />
  <Column id=email title="Email" />
  <Column id=country title="Country" />
</DataTable>


## 2. Raw Events Table

Individual user actions (Logins, Clicks, Purchases).

```sql events
select * from synthetic.raw_events
order by event_time desc
limit 500
```

<DataTable data={events} search=true rows=10>
    <Column id=event_id title="ID" />
    <Column id=event_time title="Timestamp" />
    <Column id=user_id title="User ID" />
    <Column id=event_type title="Action" />
    <Column id=revenue title="Revenue" fmt=currency />
</DataTable>
